Smart Match

Project Description
An android dating app that matches users based on personality. Users answer a list of questions when they first create their profile, and SmartMatch uses an algorithm to find the best matches.

Project Features
  • Login & Register
  • Quiz for match making
  • View matches with similar scores
  • Profile page
  • Message matches

Technical Spec
  • Android 6.0+ app
  • Programmed in Java, XML, And MySQL
  • View matches with similar scores
  • Main development environment was Android Studio and Visual Studio Code
  • Nexus 5x API 30 used as virtual device
